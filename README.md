#  Airbnb Market Analysis Dashboard (Tableau)

## Project Overview
This project analyzes Airbnb listing data to uncover pricing trends, inventory distribution, and geographic revenue hotspots. I utilized data joins to connect listing details with availability calendars to create a comprehensive market view.

## Technical Skills Demonstrated
* **Data Modeling**: Performed logical joins between `Listings` and `Calendar` data to enable time-series revenue analysis.
* **Spatial Visualization**: Developed a Zipcode-level map to visualize geographic price distribution.
* **Statistical Distribution**: Created inventory breakdowns using distinct counts to identify market saturation points.
* **Dynamic Filtering**: Implemented filters to exclude incomplete data (Nulls/0 bedrooms) for higher analysis accuracy.

## Key Findings
* Pricing increases linearly with bedroom count, with the highest average at $584.80.
* 1-bedroom units dominate the market, making up over 70% of available listings.
* Specific Zipcodes in the 98xxx range show the highest annual revenue potential.
